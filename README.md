# Portfolio Personal - Jesus Martinez 👨‍💻

Bienvenido a mi portafolio web profesional. Este proyecto es una vitrina de mis habilidades como Desarrollador Frontend, diseñado con una estética moderna, minimalista y dinámica.

🔗 **Demo en vivo**: [https://jesusmandev.github.io/frontend-portfolio-jm/](https://jesusmandev.github.io/frontend-portfolio-jm/)

## 🚀 Características Principales

- **Diseño Premium**: Interfaz elegante usando CSS puro (Vanilla CSS) con efectos de glassmorphism y animaciones suaves.
- **Multilingüe**: Soporte completo para 5 idiomas (Español, Inglés, Francés, Alemán y Portugués) con cambio dinámico y persistencia en localStorage.
- **Efectos Interactivos**:
  - Trail de cursor con partículas brillantes.
  - Efectos de Parallax y orbes de fondo dinámicos.
  - Efecto de Typewriter (máquina de escribir) para la descripción.
- **Secciones Detalladas**:
  - **Inicio**: Hero section impactante con animaciones de entrada.
  - **Sobre Mí**: Información personal y estadísticas de experiencia.
  - **Habilidades**: Visualización de tecnologías (HTML, CSS, JS, React, TypeScript, Node.js, Vite, API Consumption, Git & GitHub).
  - **Certificaciones**: Galería interactiva de logros con modales.
  - **Proyectos**: Grid de proyectos destacados con enlaces directos a demos.
  - **Contacto**: Enlaces directos a Email y WhatsApp.
- **Loader Personalizado**: Pantalla de carga interactiva con personaje animado y música de entrada.
- **Optimización SEO**: Estructura semántica y metadatos configurados.

## 🛠️ Tecnologías Usadas

- **Framework**: React 19 + TypeScript.
- **Build Tool**: Vite.
- **Estilos**: Vanilla CSS (sin frameworks).
- **Tipografía**: Google Fonts (Space Grotesk, Bebas Neue, JetBrains Mono).
- **Iconografía**: SVG personalizado inline.
- **Deployment**: GitHub Pages (via `gh-pages`).

## 📂 Estructura del Proyecto

```
src/
├── components/        # Componentes React (Loader, Navbar, Hero, About, Skills, etc.)
├── hooks/             # Custom Hooks (useLanguage, useCursor)
├── App.tsx            # Componente principal de la aplicación
├── main.tsx           # Punto de entrada de React
└── index.css          # Estilos globales
public/
├── img/               # Recursos visuales y certificados
├── CURRICULUM/        # Documentos de CV (español e inglés)
└── MUSIC/             # Audio de entrada
```

## 🔧 Instalación Local

```bash
git clone https://github.com/jesusmandev/frontend-portfolio-jm.git
cd frontend-portfolio-jm
npm install
npm run dev
```

## 🚀 Deploy a GitHub Pages

```bash
npm run deploy
```
